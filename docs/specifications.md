
# Список требований к оформлению ВКР

- [x] Размер шрифта — 14 pt
- [x] Гарнитура — Times New Roman
- [x] Междустрочный интервал - 1.5 
- [x] Выравнивание по ширине, красная строка– 1,25 см.
- [x] Поля: : левое – 30 мм, правое – 15 мм, верхнее – 20 мм, нижнее – 20 мм
- [x] Сквозная нумерация проставляется со второй страницы
- [x] Основные структурные элементы начинаются с новой страницы
- [x] Заголовки пишутся по центру прописными буквами основным шрифтом
- [x] Главы первого уровня начинаются со слова "ГЛАВА"
- [x] Содержание, введение, заключение и т. д.  не нумеруются

## Нумерация

- [x] Слово «Таблица» вместе с нумерацией указывается справа над Таблицей
- [x] Формулы нумеруются либо сквозным образом, либо в пределах главы (оба варианта можно выбрать в project_data.yaml)
- [x] Иллюстрации, рисунски, листинги нумеруются либо сквозным образом, либо в пределах главы (но нужно настраивать вручную, особенно во втором случае)

## Приложения

- [x] Название приложения пишется по центру
