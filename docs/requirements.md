
# Как и чем компилировать

Шаблон тестировался только на версии Typst 0.15.0. Поскольку программа достаточно новая, в дальнейшем возможны существенные изменения и обратная совместимость не гарантируется. 

## Локальная компиляция

Программу можно скачать для любой операционной системы на сайте [https://typst.app/open-source/#download](https://typst.app/open-source/#download). Исходный файл весит всего лишь несколько десятков мегабайт.

После установки проект можно скомпилировать, выполнив команду ```typst compile main.typ``` в корневой директории шаблона. Но лучше воспользоваться специализированной средой разработки (см. ниже).

## Среда разработки

Наиболее комфортный способ работы – использовать Visual Studio Code с официальным расширением для Typst.

**Установка:**

1. Установите VS Code (если ещё нет).
2. Откройте вкладку «Расширения» (Ctrl+Shift+X).
3. Найдите расширение Typst (автор: myriad-dreamin) и установите его.
4.  После установки откройте папку с проектом. VS Code автоматически предложит скомпилировать файл main.typ при сохранении (или используйте команду «Typst: Compile Project»).  

**Возможности:**  

+ Подсветка синтаксиса.  
+ Автодополнение для команд и меток.  
+ Быстрая компиляция с предпросмотром PDF.  
+ Встроенный вывод ошибок компиляции.  

Существуют редакторы, заточенные именно под Typst, но они, по моим ощущениям, пока что сыроваты.

## Онлайн-сервис

Если по каким-то причинам вы не можете или не хотите устанавливать Typst локально, воспользуйтесь официальным веб-приложением:
https://typst.app/

Регистрируйтесь, добавляйте файлы проекта и работайте.



## Дополнительные замечания по компиляции

+ Внешние пакеты (например, zebraw, non-unlabeled, theorion) загружаются автоматически из репозитория Typst при первой компиляции. Для работы требуется интернет.

+ Для компиляции требуется, чтобы все пути к файлам (особенно к изображениям) были корректными относительно main.typ. В шаблоне используются относительные пути, например /graphics/sine_function.png – это корень проекта, поэтому запускайте компиляцию из корня.
