
# Почему Typst?...

Преимущества перед визуальными редакторами (Word, Libre Office, Docs и др.):

* Форматирование сделано за вас
* Форматирование не может невозвратимо слететь или случайно сломаться 
* Так как стили определены в одном месте с помощью кода, их корректность легко проверяется
* Возможность подключения сторонней системы контроля версий (git и др.)
* Результат выглядит лучше: текст аккуратнее, формулы красивее

Преимущества перед Latex: 

* Компилируется быстрее и с первого раза (Latex требует 3-4 прогонок для учета всех изменений)
* Есть предпросмотр 
* Меньше весит (Десятки мегабайт против нескольких Гигабайт)
* Установка проще
* Синтаксис однороднее и предсказуемее
* Сообщения об ошибках информативнее
* Умеет больше из коробки, пакеты не конфликтуют
* Поддержка accessibility
